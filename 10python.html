제1장. 파이썬 개요 및 환경 설정

Keywords

python, numpy, pandas, 기초, 문법

1. 언어의 이해

프로그래밍 언어를 학습하기에 앞서, 우리가 일상에서 사용하는 ’언어’의 본질적인 의미와 역할을 고찰해 본다. 언어는 단순히 정보를 전달하는 수단을 넘어 사고의 틀을 형성하는 핵심적인 요소이다.

1.1 언어의 정의와 목적

언어(言語)는 인간 사회에서 정보를 교환하고 사고를 공유하기 위한 체계적인 약속이다. 학술적 관점에 따른 정의와 언어를 사용하는 주요 목적은 다음과 같다.

구분 내용
정의 생각이나 감정을 나타내고 전달하기 위해 사용하는 체계적인 상징의 집합
의사소통 개인 간의 정보, 감정, 의도를 효과적으로 교환하는 수단
사고의 도구 복잡한 개념을 논리적으로 정리하고 구체화하는 지적 기반
문화/사회 공동체의 유대감을 형성하고 세대 간 지식과 가치관을 전승하는 매체

1.2 의사전달과 통역

서로 다른 언어를 사용하는 화자 사이에서 통역사가 가교 역할을 하듯, 인간이 컴퓨터와 소통하기 위해서는 컴퓨터가 이해할 수 있는 언어와 이를 전달할 방식이 필요하다. 특히 컴퓨터는 모호한 표현을 처리하지 못하므로 매우 명확하고 구체적인 명령 체계를 사용해야 한다.

2. 프로그래밍 언어

프로그래밍 언어는 컴퓨터가 특정 작업을 수행하도록 지시하기 위해 고안된 인공적인 언어이다. 인간의 언어와 기계어 사이에서 중재 역할을 하며, 하드웨어와의 거리에 따라 저급 언어와 고급 언어로 분류된다.

2.1 실행 방식에 따른 분류

소스 코드가 컴퓨터에서 실행되는 방식은 크게 컴파일 방식과 인터프리터 방식으로 나뉜다. 파이썬은 대표적인 인터프리터 언어에 해당한다.

방식 설명 주요 언어
컴파일(Compile) 코드 전체를 기계어로 번역한 후 실행 파일을 생성. 실행 속도가 빠름. C, C++, Rust
인터프리터(Interpreter) 코드를 한 줄씩 읽어 들이며 즉시 실행. 개발 및 디버깅이 용이함. Python, JS, Ruby
혼합형(JIT) 컴파일과 인터프리팅의 장점을 결합하여 실행 성능을 최적화함. Java, C#

2.2 주요 언어별 특징

현대 소프트웨어 개발 환경에서는 목적에 따라 다양한 언어가 사용되고 있으며, 파이썬은 그중에서도 독보적인 범용성을 자랑한다.

  • C / C++: 시스템 자원 제어 및 고성능 연산(게임 엔진, 운영체제)에 적합
  • Java: 대규모 기업용 시스템 및 안드로이드 앱 개발에 널리 사용
  • Python: 간결한 문법을 바탕으로 인공지능, 데이터 과학, 업무 자동화에 최적화
  • JavaScript: 웹 브라우저의 동적 기능을 구현하는 웹 개발의 필수 언어

3. 파이썬(Python) 소개

파이썬은 “인생은 짧으니, 파이썬이 필요하다(Life is short, you need Python)”는 슬로건으로 유명할 만큼 생산성이 높은 언어이다. 본 교재는 최신 표준인 Python 3.x를 기준으로 설명한다.

3.1 파이썬의 주요 특징

파이썬은 입문자에게 친숙하면서도 전문가에게는 강력한 도구를 제공하는 이중적인 매력을 가지고 있다.

  • 간결성: 영어 문법과 유사하여 코드를 읽고 이해하기 쉬움
  • 확장성: 수많은 라이브러리와 프레임워크를 통해 다양한 분야에 즉시 적용 가능
  • 플랫폼 독립성: 운영체제(Windows, macOS, Linux)에 상관없이 동일한 코드 실행 가능

3.2 타 언어와의 비교 예제

동일한 기능을 구현할 때 파이썬의 간결함을 아래의 파일 읽기 예제를 통해 확인할 수 있다.

[예제] 텍스트 파일 내용을 읽어 화면에 출력하기

# Python 3.x 예제 코드
with open("sample.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)

4. 환경 설정 및 실습 준비

파이썬 프로그램을 작성하고 실행하기 위해서는 인터프리터 설치와 함께 적절한 개발 도구(IDE)를 선택하는 것이 중요하다.

4.1 파이썬 설치 및 확인

  1. 다운로드: python.org에서 운영체제에 맞는 3.x 버전을 다운로드한다.
  2. 주의사항: Windows 설치 시 “Add Python to PATH” 항목을 반드시 체크한다.
  3. 버전 확인: 터미널 또는 명령 프롬프트에서 아래 명령어를 입력한다.
python --version

4.2 주요 개발 환경 비교

학습자의 목적과 프로젝트 규모에 따라 적합한 도구를 선택한다.

도구 특징 추천 용도
VS Code 가볍고 확장 프로그램이 풍부한 범용 에디터 일반적인 프로그래밍, 웹 개발
PyCharm 파이썬 전용의 강력한 기능을 제공하는 전문 IDE 대규모 프로젝트, 전문 개발
Jupyter 코드와 실행 결과, 설명을 블록 단위로 관리 데이터 분석, 연구, 교육

5. [부록] 주요 용어 사전

대학생 독자들이 프로그래밍 학습 시 혼동하기 쉬운 용어들을 정리한다.

  • 인터프리터(Interpreter): 소스 코드를 기계어로 미리 번역하지 않고, 실행 시점에 한 줄씩 해석하여 결과를 출력하는 프로그램이다.
  • 라이브러리(Library): 특정 기능을 수행하는 코드들을 미리 작성하여 모아놓은 꾸러미이다. 도서관에서 책을 빌리듯 필요한 기능을 불러와 사용할 수 있다.
  • 프레임워크(Framework): 소프트웨어 개발을 위한 기본 구조(뼈대)를 제공한다. 라이브러리가 도구라면, 프레임워크는 도구가 갖춰진 작업장과 같다.
  • IDE(Integrated Development Environment): 코딩, 디버깅, 컴파일 등 개발에 필요한 모든 과정을 하나의 프로그램 안에서 처리할 수 있는 통합 개발 환경이다.

6. 제1장 요약

  1. 언어의 역할: 인간과 컴퓨터 사이의 명확한 의사소통을 위해 프로그래밍 언어가 존재한다.
  2. 파이썬의 위치: 인터프리터 방식의 고급 언어로, 간결한 문법과 높은 생산성이 특징이다.
  3. 학습 기준: 본 과정은 Python 3.x 버전을 사용하며, 이는 이전 버전(2.x)과 호환되지 않는 현대적 표준이다.
  4. 준비 사항: 파이썬 인터프리터를 설치하고, 본인의 성향에 맞는 IDE(VS Code 등)를 갖추는 것이 실습의 시작이다.
